## Dev quickstart

```bash
git clone git@github.com:milankinen/culli.git
cd culli && npm i
npm t
```

#### Creating new package

```bash
npm run newpkg
```

#### Updating dependencies

```bash 
npm run upd
```

#### Commiting changes

```bash
npm run ci
```

**ATTENTION:** Please use this command instead of `git commit`
